Create GIT Repository:
1) git init,
2) git remote add origin <link>,
3) git add .,
4) git status,
5) commit,
6) push.

Create Django Project:
1) python -m pip install Django,
2) django-admin startproject name.

Create Django app:
1) python manage.py startapp firstapp.

Run Django Project:
1) python manage.py runserver.
